# simple-ping-pong

This is a simple *Ping-Pong* game made in **Java**, if you find any bugs, please tell me about them so i may fix them.

## Controls:
### Player 1: 
 - **W** - Move paddle up
 - **S** - Move paddle down
### Player 2: 
 - **Up Arrow** - Move paddle up
 - **Down Arrow** - Move paddle down
